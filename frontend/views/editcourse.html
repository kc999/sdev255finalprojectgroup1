Searching & Filtering

Use the search bar to find courses by course name, prefix, subject area, or keywords. Filters allow you to narrow results by subject, number of credits, or availability.

Course Information

Each course listing includes: Course Prefix & Number (e.g., CS 101), Course Name, Description of course content, Number of Credits, Subject Area,

Access & Permissions

You must be logged in to make changes. If you see “Access Denied,” your account does not have permission to perform that action.

Common Issues

Course not saving? Make sure all required fields are filled
Update failed? Try refreshing the page and logging in again.
Page not loading courses? Check your internet connection.

Need More Help?
